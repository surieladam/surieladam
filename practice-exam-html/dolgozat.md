### Időtartam: 45 perc
### Pontszám: 65 pont

1. **Mit jelent a HTML rövidítés? (2 pont)**

2. **Mi a DOCTYPE deklaráció szerepe? (3 pont)**
   - _________________________________________

3. **Párosítsd össze a HTML elemeket a funkciójukkal! (5 pont)**
   ```
   1. <table>     a) Képek beillesztése
   2. <img>       b) Táblázat létrehozása
   3. <h1>        c) Táblázat fejléce/címe
   4. <caption>   d) Legnagyobb címsor
   5. <td>        e) Táblázat cellája
   ```

4. **Egészítsd ki a táblázat szerkezetét! (8 pont)**
   ```html
   <table>
     <_____>Címsor</_____>
     <tr>
       <__>Fejléc</___>
       <__>Cella</___>
     </tr>
   </table>
   ```

5. **Jelöld meg a helyes állításokat! (5 pont)**
   - [ ] A `<style>` tag a `<body>` részbe kerül
   - [ ] A `<th>` címsor cellát jelöl
   - [ ] A `colspan` attribútum cellák összevonására szolgál
   - [ ] Az `<ul>` számozott listát hoz létre
   - [ ] A `float` CSS tulajdonság képek pozicionálására használható

6. **CSS tulajdonságok - Írd le mire szolgálnak! (12 pont)**
   - margin: ______________
   - width: ______________
   - text-align: ______________
   - float: ______________

7. **Gyakorlati feladat - Hobbik bemutatása (15 pont)**
   Hozz létre egy `dolgozat.html` fájlt, amely a kedvenc hobbijaidat mutatja be! A következő elemeket kell tartalmaznia:
   - Használj megfelelő DOCTYPE deklarációt     
   - A fejlécben legyen style tag a CSS formázásokhoz   
   - Készíts egy 2x2-es táblázatot  
   - A táblázatnak legyen caption címsora   
   - Minden cellában legyen egy hobbi:  
     - Címsor (h2)
     - Kép a hobbiról (img)
     - Felsorolás (ul) legalább 3 jellemzővel (li)
   - CSS formázások:
     - A képek legyenek jobbra igazítva (float: right)
     - A képek szélessége legyen 75%
     - A cellák szélessége legyen 400px
     - A címsorok legyenek középre igazítva

_Segítség: A gyakorló fájlodban látható évszakos weboldal összes fontos HTML eleme szerepel a feladatokban!_

*A dolgozat végén ellenőrizd:*
- Minden kérdésre válaszoltál?
- A HTML kódok szintaktikailag helyesek?
- A CSS tulajdonságok pontosan vannak megadva?
